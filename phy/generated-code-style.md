# Generated Code Style

## Goals
Generated TypeScript must be deterministic, readable and stable.

## Ordering
1. Imports
2. Constants
3. Metadata
4. Class
5. Helpers
6. Exports

## Naming
SoftGenerated<Component>
SoftMetadata<Component>
SoftProxy<Component>

## Formatting
4 spaces.
LF only.
Stable import ordering.
No random ids.

## Comments
Generated files contain only a single header comment.

## Rules
No dead code.
No unused imports.
No reflection.
No string-based property names when strongly typed alternatives exist.
